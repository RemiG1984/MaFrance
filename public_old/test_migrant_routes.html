Test des Routes API - Centres Migrants

Routes par Commune (COG)

Exemples avec des COGs de grandes villes:

/api/migrants/commune/75056 (Paris) /api/migrants/commune/69123 (Lyon) /api/migrants/commune/13055 (Marseille) /api/migrants/commune/31555 (Toulouse) /api/migrants/commune/59350 (Lille)

Routes par Département

Exemples avec différents départements:

/api/migrants/departement/75 (Paris) /api/migrants/departement/69 (Rhône) /api/migrants/departement/13 (Bouches-du-Rhône) /api/migrants/departement/59 (Nord) /api/migrants/departement/93 (Seine-Saint-Denis) /api/migrants/departement/2A (Corse-du-Sud) /api/migrants/departement/971 (Guadeloupe)

Routes avec Pagination

Tests de pagination sur les départements:

/api/migrants/departement/75?page=1&limit=10 /api/migrants/departement/69?page=1&limit=5 /api/migrants/departement/13?page=2&limit=20

Statistiques

/api/migrants/stats

Tests d'Erreurs

Ces liens devraient retourner des erreurs de validation:

/api/migrants/commune/invalid (COG invalide) /api/migrants/departement/99 (Département invalide) /api/migrants/departement/75?page=0 (Page invalide) /api/migrants/departement/75?limit=2000 (Limite trop élevée)

Notes:

  • • Les réponses seront au format JSON
  • • Les erreurs de validation retourneront un code 400
  • • Les ressources non trouvées retourneront un code 404
  • • Ouvrez la console développeur pour voir les détails des réponses
